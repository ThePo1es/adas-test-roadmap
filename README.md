# 자율주행 시험(ADAS Test Engineer) 4주 부트캠프

> 목표 직무: 현대자동차 자율주행/주차 시스템 시험 (실차 검증, Test Case 개발, 성능육성)
> 대상: 자동차IT공학 2학년, 코딩은 LLM 도움 없이는 기초 수준
> 기간: 4주 × 주 7일 × 5시간 (6일 학습 + 1일 복습/정리)

## 왜 이 커리큘럼인가

시험(검증) 직무의 실제 업무는 ①요구사양 분석 → ②Test Case 설계 → ③실차/시뮬레이션 시험 → ④데이터 분석 → ⑤리포트다.
그래서 이 로드맵은 "코딩 기초 → 데이터 분석 → 차량 도메인 → 시험 시나리오 시뮬레이션 캡스톤" 순서로 간다.
4주 뒤 결과물: **AEB(자동긴급제동) 시험 시나리오를 직접 설계하고 파이썬으로 시뮬레이션·자동판정하는 포트폴리오 저장소.**

## 절대 규칙

1. **LLM 코드 생성 금지.** 검색(공식 문서, Stack Overflow)은 허용. LLM은 "개념 질문"과 "내 코드 리뷰 요청"에만 사용.
2. 매일 커밋. 커밋 메시지에 그날 배운 것 1줄.
3. 막힌 문제는 30분 버티고, 그래도 안 되면 질문하고 넘어간다 (기록 남기기).
4. 매주 7일차에 `TIL.md`(Today I Learned) 주간 회고 작성.
