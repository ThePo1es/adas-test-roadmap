# Week 1 — 라즈베리파이 + Python + Git + GPIO

**목표:** 파이썬 기본기 + 라즈베리파이 GPIO로 실제 부품(LED/버튼/초음파)을 제어. 손으로 만지며 코딩에 익숙해진다.
**참고 문제풀이:** 코드업(codeup.kr) 기초 100제 / 프로그래머스 Lv.0 (백준은 2026.4 종료 후 부활 예정)
**준비물:** 라즈베리파이, LED+220Ω, 푸시버튼, HC-SR04, 브레드보드, 점퍼선

| 시간 | 할 일 |
|------|------|
| 0:00~1:00 | 라즈베리파이 OS 확인/업데이트(`sudo apt update && upgrade`), 파이썬/기본 도구 점검. VSCode 원격(SSH) 또는 Pi에서 직접 편집 |

| 2:00~4:00 | **첫 하드웨어:** LED+저항을 GPIO에 연결(전원 끄고 배선→사진→확인→전원). `gpiozero`로 1초 점멸 → 점멸 주기 바꾸기 → 버튼 누르면 켜지게 |
| 4:00~5:00 | 배선 사진 + `src/blink.py` 커밋, notes-day1.md |
